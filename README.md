# COVIDHostAnalysis
Code for analyses as presented in "Early Type II IFN Limits SARS-CoV-2 Replication in Humans"

Datafiles called in the provided code can be found at:

Winnett, A., Tabachnikova, A., Chen, J., Greene, K., Romano, A., Pei, X. P., Cooper, M., Silva, J., Carter, A., Jiang, J.,
Kong, Y., Roos, M., Middle, C., Zhang, H., Thomson, M., Kuersten, S., Booher, K., Iwasaki, A., & Ismagilov, R. F. (2026). 
Early Mucosal Type II Interferon Limits SARS-CoV-2 Replication in Humans [Data set]. CaltechDATA. https://doi.org/10.22002/krbp7-h5z17

Data has been deidentified, including Participant IDs and exact ages, in accordance with Caltech IRB requirements. 

For questions, please contact Alexander Viloria Winnett (awinnett@caltech.edu, aviloriawinnett@mednet.ucla.edu).
